## 2.2 代码块与公式
### （1）Python代码块
**代码功能：读取销售数据并计算平均销售额、订单量与客单价**

```python
import pandas as pd

# 读取CSV格式的销售数据
data = pd.read_csv("sales_data.csv")

# 计算核心指标的均值
avg_sales = data["销售额（万元）"].mean()
avg_orders = data["订单量（单）"].mean()
avg_price = data["客单价（元）"].mean()

# 打印结果
print("=== 销售数据核心指标均值 ===")
print(f"平均销售额：{avg_sales:.2f}万元")
print(f"平均订单量：{avg_orders:.0f}单")
print(f"平均客单价：{avg_price:.2f}元")

```
### （2）
#### ① 算术均值公式
$\bar{x} = \frac{1}{n} \sum_{i=1}^{n} x_i$

**公式说明**：计算一组数据的算术平均值，反映数据的集中趋势，是描述性统计最基础的指标。其中 $n$ 为样本总数，$x_i$ 为第 $i$ 个样本值。
#### ② 总体方差公式
$\sigma^2 = \frac{1}{n} \sum_{i=1}^{n} (x_i - \bar{x})^2$

# 三、综合实践题 
# 某电商平台2026年1-4月月度销售数据分析报告
## 一、数据概况
本次分析选取某电商平台2026年1月至4月的核心销售数据，旨在评估平台近期销售表现，识别增长趋势与潜在问题，为后续运营决策提供数据支持。分析数据涵盖**总销售额**、**订单量**、**客单价**及**主要品类销售额占比**四个维度，数据来源于平台内部销售管理系统。
## 二、数据分析结果

### 2.1 整体销售趋势
2026年1-4月平台整体销售呈现**稳步增长**态势，总销售额从1月的128.5万元增长至4月的186.2万元，累计增长率达44.9%。具体月度销售数据如下表所示：

| 月份 | 总销售额(万元) | 订单量(单) | 客单价(元) |
|------|----------------|------------|------------|
| 1月  | 128.5          | 3210       | 400.3      |
| 2月  | 156.8          | 3895       | 402.6      |
| 3月  | 172.3          | 4256       | 404.8      |
| 4月  | 186.2          | 4580       | 406.6      |

从数据可以看出：
- 订单量逐月递增，4月订单量较1月增长42.7%
- 客单价保持小幅稳定上升，4个月累计增长1.6%
- 销售额增长主要由**订单量增长**驱动
### 2.2 品类销售结构分析
4月份各主要品类销售额占比情况如下：
- 电子产品：42.3%
- 服装配饰：28.7%
- 家居用品：16.5%
- 食品饮料：8.2%
- 其他品类：4.3%

电子产品和服装配饰合计贡献了超过70%的销售额，是平台的核心品类。其中电子产品销售额环比增长12.5%，表现最为突出。
### 2.3 销售额环比增长率计算
为了量化月度增长情况，我们使用环比增长率公式进行计算：

$r = \frac{本月销售额 - 上月销售额}{上月销售额} \times 100\%$

**公式说明**：环比增长率用于比较相邻两个周期的增长情况，能够直观反映短期销售变化趋势。
